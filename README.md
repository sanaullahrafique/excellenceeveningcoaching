# Excellence Evening Coaching Website

A modern, animated website for Excellence Evening Coaching institute built with Tailwind CSS 4.

## Features

- **Responsive Design**: Works perfectly on all devices
- **Smooth Animations**: Custom CSS animations and transitions
- **Modern UI**: Clean and professional design
- **Interactive Elements**: Hover effects and smooth scrolling
- **Mobile-Friendly**: Responsive navigation and layout

## Coaching Services

- 9th Class coaching
- 10th Class coaching  
- 1st Year (Pre-Medical & Pre-Engineering)
- 2nd Year with MDCAT/ECAT preparation

## Key Features Highlighted

- Experienced teaching staff
- Modern educational environment
- In-time completion of courses
- Extended approach for every MDCAT topic
- A+ test series after syllabus completion
- Monthly test system
- Individual focus on every student
- Exemplary discipline
- Separate classes for boys and girls

## Development

To run the development server:

```bash
npm run dev
```

This will watch for changes in `tw.css` and automatically compile to `style.css`.
